# RFID-Servo-Security-System-with-RGB-Feedback
A secure electronic lock system that uses an RFID reader to grant access. Visual feedback is provided via an RGB LED: Red indicates access denied or locked status, while Green indicates access granted.🛠 Hardware RequirementsController: Arduino UnoModule: MFRC522 RFID ReaderActuator: SG90 Servo MotorIndicator: RGB LED (Common Cathode)Misc: Breadboard, Jumper Wires, RFID Tag/Card🔌 Wiring/Circuit TableComponentArduino PinRC522 SDA10RC522 SCK13RC522 MOSI11RC522 MISO12Servo Signal9RGB Red Pin5RGB Green Pin6🚀 How-to/SetupLibrary: Install MFRC522 and Servo from the Arduino Library Manager.Identify: Run the "DumpInfo" example sketch in the MFRC522 library to get your card's UID.Configure: Copy your UID into the authorizedUID array in the main code.Feedback Logic: The code uses digitalWrite to switch the LED pins; Red is active by default (Locked), and Green triggers only when the UID matches.📝 License/NotesFeel free to remix this for your own hardware builds!
